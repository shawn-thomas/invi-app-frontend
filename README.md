## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/shawn-thomas/invi-app-frontend.git
   ```
2. Install NPM packages
   ```sh
   npm install --legacy-peer-deps
   npm start
   ```
